# PC Ansible

Settings for PC setup in MacOS.

## Install Homebrew

```shell
# install xcode
xcode-select --install

# install homebrew from bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```

## Install Ansible

```shell
# install ansible
brew install ansible ansible-lint
```
